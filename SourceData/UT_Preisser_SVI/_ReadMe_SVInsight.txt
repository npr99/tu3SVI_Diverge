How to obtain SVInsight Data

Go to the GitHub Repo: https://github.com/mdp0023/SVInsight

Follow instructions for setting up SVInsight python package.

Review SVInsight Documentation https://mdp0023.github.io/SVInsight/index.html

Follow instructions to set up model for Texas, 2020 data for tracts and block groups.

Code example for Southeast Texas:
https://mdp0023.github.io/SVInsight/Examples/SVInsight_example_SETx.html
