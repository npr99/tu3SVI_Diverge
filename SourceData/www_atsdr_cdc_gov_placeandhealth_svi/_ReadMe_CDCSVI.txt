How to obtain CDC SVI data

1. Go to the website: https://www.atsdr.cdc.gov/placeandhealth/svi/data_documentation_download.html
2. Select year= 2020, Geography = Texas, Geography Type = Census Tracts
3. File Type = CSV File (table data)
4. Click 'Go'
